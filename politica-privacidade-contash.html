Política de Privacidade - ContasH

Última atualização: 09 de julho de 2026

1. Introdução

A presente Política de Privacidade descreve como o aplicativo ContasH ("aplicativo") coleta, usa e protege as informações dos seus usuários. Ao utilizar nosso aplicativo, você concorda com as práticas descritas nesta política.

2. Dados Coletados

O aplicativo ContasH coleta apenas os seguintes dados:

Dado Finalidade Armazenamento
Nome do jogador Exibição no jogo e ranking Localmente no dispositivo
Pontuação e medalhas Ranking de melhores jogadores Localmente no dispositivo

Importante: Todos os dados são armazenados apenas localmente no seu dispositivo. Nenhum dado é transmitido para servidores externos.

3. Dados que NÃO Coletamos

  • Dados de localização
  • Informações de contato (e-mail, telefone)
  • Dados de uso da internet
  • Informações do dispositivo
  • Dados de analytics ou rastreamento
  • Informações de pagamento

4. Permissões do Aplicativo

O aplicativo ContasH NÃO solicita nenhuma permissão especial do dispositivo, como:

  • Acesso à internet
  • Acesso à câmera
  • Acesso à localização
  • Acesso aos contatos
  • Acesso à galeria de mídia

5. Compartilhamento de Dados

Não compartilhamos, vendemos ou transmitimos seus dados pessoais para terceiros. Todos os dados permanecem exclusivamente no seu dispositivo.

6. Segurança dos Dados

Utilizamos o armazenamento seguro do Android (SharedPreferences) para manter seus dados protegidos. Como os dados não saem do dispositivo, o risco de exposição é mínimo.

7. Retenção de Dados

Os dados são mantidos no dispositivo enquanto o aplicativo estiver instalado. Você pode apagar seus dados a qualquer momento desinstalando o aplicativo ou limpando os dados pelo menu de configurações do Android.

8. Privacidade de Crianças

O aplicativo ContasH é adequado para todas as idades, incluindo crianças. Não coletamos dados pessoais de crianças e não há interação com serviços externos.

9. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Quaisquer alterações serão refletidas na data de "Última atualização" acima.
